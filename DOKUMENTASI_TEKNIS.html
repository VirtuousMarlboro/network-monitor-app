1. Gambaran Umum Sistem

Network Monitor App adalah aplikasi web untuk memantau perangkat jaringan secara real-time menggunakan protokol ICMP Ping dan SNMP. Aplikasi ini menyediakan dashboard monitoring, ticketing system, dan notifikasi multi-channel.

Fitur Utama

Fitur Deskripsi
✅ Real-time Monitoring Ping otomatis setiap 10 detik dengan Server-Sent Events (SSE)
✅ SNMP Traffic Monitoring Monitoring bandwidth interface dengan dukungan 32/64-bit counters
✅ Ticketing System Auto-ticket untuk host down dengan delay 2 menit
✅ Multi-channel Notifications Telegram, Push Notifications, Webhooks
✅ Geo-location Map Peta interaktif dengan Leaflet.js
✅ Security Hardened WAF, CSP, Rate Limiting, HTTPS, Session Management

5. Prosedur Penggunaan

5.1 Instalasi & Konfigurasi

# 1. Clone repository
git clone <repository-url>
cd network-monitor-app

# 2. Install dependencies
npm install

# 3. Setup environment (.env)
cp .env.example .env
# Edit .env, set SESSION_SECRET minimal 32 karakter

# 4. Jalankan aplikasi
npm start
# Akses: http://localhost:3000

5.2 Login Pertama Kali

  1. Buka browser ke http://localhost:3000
  2. Login dengan kredensial default: admin / ChangeThisStrongPassword123!
  3. Wajib ganti password dengan minimal 8 karakter (huruf besar, kecil, angka, simbol)

5.3 Menambah Host

  1. Klik tombol "+ Host" di header
  2. Isi form:
    • IP/Hostname: Alamat target (contoh: 192.168.1.1)
    • Nama: Label deskriptif
    • CID: Customer ID (opsional)
    • Grup: Pilih group (opsional)
    • Lokasi: Toggle untuk menambah koordinat peta
  3. Klik "Simpan"

5.4 Mengaktifkan SNMP Monitoring

  1. Edit host yang sudah ada
  2. Enable SNMP Monitoring
  3. Set Community String (default: public)
  4. Pilih SNMP Version (2c recommended)
  5. Klik Scan Interface untuk auto-detect
  6. Pilih interface untuk monitoring
  7. Simpan perubahan

5.5 Mengelola Tiket

  1. Tab "Tiket" untuk melihat semua tiket
  2. Tiket otomatis dibuat jika host down > 2 menit
  3. Filter berdasarkan status/host/tanggal
  4. Klik tiket untuk update status atau tambah komentar
  5. Export ke Excel dengan tombol "Export"

5.6 Konfigurasi Notifikasi Telegram

  1. Buat bot Telegram via @BotFather
  2. Dapatkan Bot Token dan Chat ID
  3. Masuk ke Settings (ikon gear) > Tab Telegram
  4. Masukkan token dan chat ID
  5. Klik "Test" untuk verifikasi

6. Evaluasi UI: 8 Golden Rules

Ben Shneiderman's 8 Golden Rules of Interface Design
Evaluasi UI berdasarkan prinsip desain antarmuka yang diakui secara internasional.
# Rule Implementasi Status
1 Strive for Consistency Desain konsisten: warna, ikon, button styles sama di seluruh aplikasi. Navigasi tab konsisten. ✅ Terpenuhi
2 Seek Universal Usability Responsive design, dark/light mode, keyboard shortcuts, accessible color contrast. PWA support. ✅ Terpenuhi
3 Offer Informative Feedback Toast notifications untuk semua aksi, loading states, SSE real-time updates, sound alerts untuk host down/up. ✅ Terpenuhi
4 Design Dialogs for Closure Modal dengan header/body/footer jelas, tombol Cancel/Save eksplisit, konfirmasi delete. ✅ Terpenuhi
5 Prevent Errors Validasi input (IP, password strength), konfirmasi delete. Bisa ditingkatkan: inline validation saat typing. ⚠️ Sebagian
6 Permit Easy Reversal Tidak ada fitur undo/restore setelah delete. Backup service ada tapi tidak ekspos ke UI. ⚠️ Kurang
7 Support Internal Locus of Control User mengontrol penuh: tambah/edit/hapus host, manual ping, filter/search, theme toggle. ✅ Terpenuhi
8 Reduce Short-term Memory Load Stats summary selalu visible, host cards dengan status warna, search/filter untuk navigasi cepat. ✅ Terpenuhi

Rekomendasi Perbaikan

  1. Error Prevention (Rule 5):
    • Tambah inline validation real-time pada form
    • Disable tombol save jika form invalid
    • Tambah auto-suggest untuk IP address
  2. Easy Reversal (Rule 6):
    • Implementasi "Undo Delete" dengan soft-delete dan grace period
    • Tampilkan tombol restore di UI untuk item yang baru dihapus
    • History/versioning untuk perubahan konfigurasi
  3. Accessibility:
    • Tambah ARIA labels pada interactive elements
    • Keyboard navigation untuk semua modal
    • Skip-to-content link
